# Liquid Glass

![Liquid Glass demo](public/Screenshot%20from%202026-03-14%2003-50-05.png)

A small React demo with a **liquid glass** effect: draggable cards with WebGL refraction, chromatic aberration, and a subtle wobble on drag.

![Screenshot 2](public/Screenshot%20from%202026-03-14%2003-50-33.png)
![Screenshot 3](public/Screenshot%20from%202026-03-14%2003-49-59.png)

## Stack

- React 18, Vite 7
- WebGL (custom vertex/fragment shaders) for background refraction
- [interact.js](https://interactjs.io/) for drag-and-drop
- Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```
